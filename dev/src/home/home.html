系统使用说明

综述

本项目是 数据类型配置、 流程配置、 表单配置、 人员权限配置、 部门配置、 站点配置、 数据字典配置 的前端样本工程。工程仅仅包含前端，数据结构为标准的JSON结构。在实际使用时候需要转换为标准的ER结构。

后文会大量的使用Json结构，先介绍JSON结构相关意义：

JSON

Map结构、表结构：

    const data = {key1:{
        in_key1:'value1'
        in_key2:'value2'
    }}

key1是map的key，后面的{}是map的数据，JS中可以通过data['key1']或data.key1获取数据。

如果是做一行表数据,key1是主键,in_key1、in_key2是字段，value1、value2是对应的数据。
